# PlayWales Backend

This is the Flask backend for the PlayWales application.

## Setup

1. Ensure you have a virtual environment set up and dependencies installed:
   ```bash
   pip install -r requirements.txt
   ```

2. Seed the database with mock data:
   ```bash
   python seed.py
   ```

## Running the Application

```bash
python app.py
```

## Running Tests

```bash
python -m pytest
```
